# Ubuntu SOC Validation and Backup Automation

## Overview
This repository contains a Computer Network Defence project developed during my Bachelor's degree in Cyber Security at Noroff University College.

The project implements an automated Bash script that performs system validation, integrity checking, secure backups, and SOC-style reporting on Ubuntu servers.

## Features
- Automated system validation
- File integrity verification using SHA256
- GPG signature validation
- Secure backup creation
- SSH-based file uploads
- Automated reporting
- Cron-based scheduling
- Error handling and logging

## Technologies Used
- Bash
- Ubuntu Linux
- OpenSSH
- GPG
- Cron
- SHA256
- Tar/Gzip
